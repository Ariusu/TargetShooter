東京電機大学所属　新村健人

～～操作可能機種～～
PCのみ（キーボードとマウス）

～～起動方法～～
TargetShooter.exeのアプリケーションをダブルクリックすることで起動することができます。
「Windows によって PC が保護されました」が表示された場合、詳細情報をクリックし、実行するを押すことで起動ができます。


～～操作説明～～
メニュー画面では制限時間を40秒か60秒を選ぶことができます。

プレイする際にはWASDで移動。マウスを動かすことでカメラ視点操作ができます。

左クリックで射撃、スペースでジャンプができます。

ステージは4つあります。敵はステージごとに5体です。

敵の位置が分かりづらいかもしれませんが、発見するのもゲームの一つの内容なのでご了承ください。

ゲームオーバーではスペースキーを押すと3秒後にスタート画面が表示されるようになっています。

全画面共通でエスケープキーを押すとゲームが終了するようになっています。

ゲームクリアをした場合はエスケープキーを押してゲームを終了するか、スペースキーでリスタートしてください。
